| About MIT Food Computer project

MIT has it's own version of food computer (currently 2.0) which is open source and documetation is getting better. The project is quite complex and some components are quite expensive and difficult to get in UK

Concept

The idea of food computer is to have automated system to created controled environment for growing plants. All necessary factors for plants: light, water, air, humidity and temperature would be controled by Arduino which would be connected to appropriated devices. There would be set of sensors capturing real time data of the current environment status.

Hydrophonics

OK, so this is not that much about electronics but it's very important part of the project. Choosing the right hydrophonic system and seting it all up required some design thinking too. The system is combination of Ebb and flow and floating platfrom. I have chosen this approach to have better control over the water level as I can control the overflow and using the airstones to make sure the roots are getting enough o2.

Learnings so far

At first I tried to follow MIT food computer guide, but the complexity was just overwhelming and very frustrating to get started. I abbandoned the MIT version and started my own with very simple features. I am slowly improving the version 1.0 while actually growing plants.

Future plans

Currently I am working on adding sensors. The idea is to connect the sensors to wifi and send over to my website. I've been playing with ESP8266 wifi module which is basically arduino with wifi.

Improved version 2.0

After using the Food Computer for a few month I have bumped into a few issue: 1. Cheking the plants by lifting the box was quite effortful. 2. There was too much humidity inside. 3. Too many cables were annoying when positioning the pannel with electronics.

Adding Fan

To decrease the humidity I've added fan and programmed arduino to turn it one twice per day. After adding the humidity and temperature sensor I found out that this wasn't efficient enough so I set the fan to turn on every hour for 10 minutes. The humidity is increasing fairly fast.

Old computer fan is attached in reverse so the air is not blowing directly to one spot.
